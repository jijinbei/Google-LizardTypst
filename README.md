# LizardTypst

A Google Slides extension that allows you to embed SVG images in your slides.
Inspired by [IguanaLaTeX](https://www.jonathanleroux.org/software/iguanatex/).

## Features

- Real-time rendering

## Development

### Setup

```shell
npm install -g @google/clasp
clasp login
```

### Deploy to Google Apps Script

```shell
clasp push
```
