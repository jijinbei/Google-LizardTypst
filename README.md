# LizardTypst for Google Slides

A Google Slides add-on to embed beautiful formulas and text generated by [Typst](https://typst.app/).
Inspired by [IguanaLaTeX](https://www.jonathanleroux.org/software/iguanatex/).

## Features

- **Embed from Typst**: Directly insert formulas, text, and diagrams written in Typst into your slides as high-quality images.
- **No Local Installation Needed**: Powered by the incredible [typst.ts](https://github.com/Myriad-Dreamin/typst.ts) library, which runs Typst entirely in your browser. No need to install anything on your computer.
- **Real-time Preview**: Instantly see the rendered output of your Typst code as you type.
- **Equation History**: Quickly access and reuse up to 20 of your most recent entries.
- **Detailed Error Reporting**: Get clear feedback on Typst compilation errors to help you fix your code quickly.

## Development

### Setup

```shell
npm install -g @google/clasp
clasp login
```

### Deploy to Google Apps Script

```shell
clasp push
```
